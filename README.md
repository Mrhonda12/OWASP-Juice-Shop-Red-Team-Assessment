## Overview
This project documents a Red Team web application security assessment conducted against the intentionally vulnerable OWASP Juice Shop application. The assessment focuses on identifying and testing OWASP Top 10 vulnerabilities in a controlled lab environment.

## Target Application
- OWASP Juice Shop (Docker-based local deployment)
- Accessed via http://localhost:3000

## Methodology
- Manual web application testing
- Input validation testing
- Authentication testing
- Client-side inspection using browser developer tools

## Tools Used
- Firefox (Kali Linux)
- Browser Developer Tools
- Docker
- Kali Linux

## Skills Demonstrated
- Web application security testing
- OWASP Top 10 knowledge
- Red Team methodology
- Security documentation

## Disclaimer
All testing was performed in a local, intentionally vulnerable lab environment for educational and defensive security purposes only.
